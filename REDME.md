# Quebrada Que Vence

Sistema Web desenvolvido para gerenciar o projeto social **Quebrada Que Vence**, reunindo apoiadores, voluntários, empresas parceiras e administradores em uma única plataforma.

---

# 🎯 Objetivo

Desenvolver uma plataforma moderna, gratuita e escalável para facilitar a gestão do projeto social Quebrada Que Vence.

O sistema permitirá:

- Cadastro de apoiadores
- Cadastro de voluntários
- Cadastro de empresas parceiras
- Gestão de contatos (CRM)
- Dashboard com indicadores
- Mapa dos cadastrados
- Relatórios automáticos
- Integração com WhatsApp
